# ระบบจัดการหอพัก

โปรเจคนี้เป็นระบบจัดการห้องพักสำหรับหอพัก/อพาร์ตเมนต์ เขียนด้วย PHP และ MySQL ใช้สำหรับ:

- เพิ่มห้องพักใหม่
- แก้ไขข้อมูลห้องพัก
- ลบข้อมูลห้องพัก
- แสดงรายการห้องพักทั้งหมด

## โครงสร้างโปรเจค

- `domi1/`
  - `css/style.css` - ไฟล์สไตล์สำหรับหน้าตาเว็บ
  - `database/dormitory (1).sql` - สคริปต์สร้างฐานข้อมูลและข้อมูลตัวอย่าง
  - `file/`
    - `add_room.php` - ฟอร์มเพิ่มห้องพัก
    - `connect.php` - ตั้งค่าการเชื่อมต่อฐานข้อมูล
    - `delete_room.php` - ลบข้อมูลห้องพัก
    - `edit_form.php` - ฟอร์มแก้ไขข้อมูลห้องพัก
    - `index.php` - หน้าหลักแสดงรายการห้องพัก
    - `update_room.php` - อัปเดตข้อมูลห้องพักหลังแก้ไข
  - `image/` - เก็บรูปภาพหรือไฟล์สื่อที่ใช้ในโปรเจค

## วิธีติดตั้งและใช้งาน

1. วางโฟลเดอร์ `domi1` ไว้ในโฟลเดอร์ของเซิร์ฟเวอร์ PHP เช่น `htdocs` ของ XAMPP
2. เปิด phpMyAdmin แล้วนำเข้าไฟล์ฐานข้อมูล `database/dormitory (1).sql`
3. ตรวจสอบและแก้ไขค่าการเชื่อมต่อฐานข้อมูลใน `file/connect.php`
4. เปิดเว็บเบราว์เซอร์และเข้าไปที่:

   `http://localhost/domi1/file/index.php`

## ตัวอย่างการใช้งาน

- หน้าแรกจะแสดงรายการห้องพักทั้งหมด
- คลิก "เพิ่มห้องพัก" เพื่อไปยังหน้า `add_room.php`
- ในหน้ารายการห้องพัก จะมีปุ่มแก้ไขและลบสำหรับแต่ละรายการ
- คลิกปุ่มแก้ไขจะไปยังหน้า `edit_form.php` เพื่อแก้ไขข้อมูลแล้วบันทึกด้วย `update_room.php`

## ตัวอย่างโค้ดสำคัญ

ไฟล์ `file/connect.php` ควรมีการตั้งค่าประมาณนี้:

```php
<?php
$host = 'localhost';
$user = 'root';
$password = '';
database = 'ชื่อฐานข้อมูลของคุณ';
$conn = mysqli_connect($host, $user, $password, $database);
if (!$conn) {
    die('เชื่อมต่อฐานข้อมูลไม่สำเร็จ: ' . mysqli_connect_error());
}
?>
```

## หมายเหตุ

- หากต้องการใช้งานจริง ให้ตั้งค่าชื่อฐานข้อมูลและรหัสผ่านให้ถูกต้อง
- ไฟล์ `database/dormitory (1).sql` มีตัวอย่างข้อมูลสำหรับทดสอบ
- ถ้าต้องการปรับหน้าตาเว็บให้สวยขึ้น สามารถแก้ไข `css/style.css`
